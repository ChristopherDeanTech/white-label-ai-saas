# SaaS Website Project

This is a web application currently being developed in VS Code and backed up on GitHub.

## Features Planned
- User authentication
- Stripe payment integration
- MongoDB database connection
- Calendly integration
- OpenAI integration
- Contact forms
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- MongoDB

## Project Status
This project is currently a work in progress.

## Version Control Workflow
- `main` = stable code
- `dev` = testing and development
- `feature-*` = individual features

## Notes
Sensitive values such as API keys are stored in `.env` and are not pushed to GitHub.
